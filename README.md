## 기능

- 날씨 API를 이용한 날씨 구현 (OpenWeatherMap)
- 시계 JavaScript DAte() 이용
- search 기능 Google Linked 해서 구현
- todoList View hover, scroll 구현 ('click' event 발생시 todoList Link로 이동/~ing)
- card hover, click event 구현
